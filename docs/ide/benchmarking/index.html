    Бенчмарки IDE #

    This text contains Avito specific details

    Используем Performance testing plugin.

    Чтобы плагин работал корректно на macOS, необходимо разрешить Android Studio/IntelliJ IDEA доступ к функциям Accessibility системы. Для этого:

    1. Открываем System Preferences > Security & Privacy
    2. Выбираем вкладку Privacy
    3. В списке слева находим пункт Accessibility
    4. Ставим галочку в списке справа для нужных приложений

    Как попробовать? #

    1. Установи плагин
    2. Запусти Help > Diagnostic > Execute Performance Script…
    3. Выбери готовый сценарий из /benchmarks/ide и вставь содержимое скрипта в открывшееся диалоговое окно (содержимое выпадающего списка Please select scenario: не имеет значения)

    Как проверить гипотезу и измерить разницу? #

    Используем в ручном режиме:

    1. Прогони бенчмарк несколько раз до внесения изменений в проект
    2. Сохрани результаты. Они отобразятся в диалоге после прогона бенчмарка
    3. Внеси изменения в проект
    4. Прогони бенчмарк несколько раз, сравни результаты.

    Что еще? #

    • Плагин умеет записывать YourKit snapshot с детальной информацией о всех вызовах методов.
